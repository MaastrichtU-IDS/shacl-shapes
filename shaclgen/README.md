
Generate SHACL shape from RDF instances using [SHACLGEN](https://github.com/uwlib-cams/shaclgen)

Install: 

```bash
pip install shaclgen
```

Run:

```bash
shaclgen dataset.ttl -ns https://w3id.org/um/fair-metadata fairmeta
```

Resulting SHACL shapes for example `schema:Dataset` in `dataset.ttl` (RDF generated by https://maastrichtu-ids.github.io/fair-metadata-wizard/ in JSON-LD, then converted to Turtle using https://www.easyrdf.org/converter)
